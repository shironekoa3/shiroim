JdbcTemplate 概述

它是 Spring 框架中提供的一个对象，是对原始繁琐的 jdbc api 对象的简单封装。 Spring 框架为我们提供了很多的操作模板类，例如: 操作关系型数据的 JdbcTemplate 和 HibernateTemplate，操作 nosql 数据库的 RedisTemplate，操作消息队列的 JmsTemplate 等等。
