SSR V2ray 使用教程

发表于2020-10-15|更新于2020-10-15|教程
|字数总计:534|阅读时长:1分钟

Windows 客户端(Netch)

Windows 端推荐使用 Netch，支持 SSR、V2ray、Trojan，且使用简单，还能加速游戏。

程序项目地址：https://github.com/NetchX/Netch
下载地址(蓝奏云)：https://www.lanzoux.com/ixv4Ehg7zsd

  1. 下载 Netch 客户端，解压后以管理员身份运行 Netch.exe 程序。

  2. 打开程序后，选中 “订阅” > “管理订阅链接”。

  3. 粘贴订阅链接到左下角的链接，备注随便填，点击“保存/修改”。

  4. 到主界面点击 “从订阅链接更新服务器” 即可获取到节点。

  5. 主界面切换可用服务器，模式选择 “[网页代理] 绕过局域网和中国大陆”，再点击右下角启动即可。

Android 客户端(SSR)

此客户端为 SSR 协议客户端。

程序项目地址：https://github.com/shadowsocksrr/shadowsocksr-android
程序下载地址(蓝奏云)：https://www.lanzoux.com/iQzFhhg88la

  1. 下载 SSR 客户端。

  2. 打开 ShadowsocksR(R) 点击左上角的「ShadowsocksR」进入配置文件管理页面。点击右下角的加号按钮，在弹出的选项中选择「添加/升级 SSR 订阅」。

  3. 点击选择您需要的节点，随后往下划到功能设置处，点击「路由」将其更改为 绕过局域网及中国大陆地址。随后往上划到顶部，点击右上角小飞机即可成功连接。

Android 客户端(v2rayNG)

此客户端为 VMess 协议客户端。

程序项目地址：https://github.com/2dust/v2rayNG/
程序下载地址(蓝奏云)：https://www.lanzoux.com/iBYsLhg8d0j

  1. 下载 v2rayNG 客户端。

  2. 打开 V2rayNG 点击左上角的菜单图标打开侧边栏，随后点击 订阅设置。点击右上角的加号按钮，在新页面的备注中填写本站名称，地址输入框中粘贴订阅链接并点击右上角的 √。

  3. 再次从侧边栏进入 设置 页面，点击 路由模式 将其更改为 绕过局域网及大陆地址。

  4. 随后从侧边栏回到 配置文件 页面，点击右上角的省略号图标选择更新订阅。

  5. 点击选择您需要的节点，点击右下角的按钮即可连接。
