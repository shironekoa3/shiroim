使用 iperf3 进行局域网测速

发表于2023-05-20|更新于2023-05-20|教程
|字数总计:730|阅读时长:2分钟

前言

在使用电脑或手机时，经常会遇到多设备文件互相传输的情况。

一般，我们可以使用 SMB、FTP 等协议来进行文件共享传输，但是速度上却有所不同。

比如在 Windows 上直接共享文件是使用的 SMB 协议，然而在手机端使用软件来下载电脑上共享的文件时，根据软件所使用的 SMB 版本(SMB1/SMB2/SMB3)，会有不同的传输速度，差距较大。

要想知道两个设备之间的最大传输速度，可以使用 iperf 工具来进行局域网测速。

下载

使用 iperf 需要一个服务端和一个客户端，我们可以在 Windows 电脑上运行服务端，手机上运行客户端来进行连接并测速。

Windows 版本直接在官网下载，手机上推荐使用 PingTools Pro 软件，内置 iperf 客户端。

iperf 官网下载地址：https://iperf.fr/iperf-download.php#windows

如果官方无法访问，也可以点击以下按钮进行下载：

iperf-3.1.3-win64 PingTools Pro
